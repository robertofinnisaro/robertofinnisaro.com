This is a guide on how to install ROS-Noetic
It is recommended to use Ubuntu 16.04, 18.04 or 20.04 as the base operating system for ROS to be installed on.
In this guide it will run through how to download and install ROS onto a virtual machine running Ubuntu 20.04.

# Prerequisites
Begin by downloading the correct ISO image for Ubuntu, for this we will be using Ubuntu 20.04 from the offical Ubuntu website: releases.ubuntu.com/focal/

At this point it is also advised to download a VMM (Virtual Machine Manager), for this we will be working off Windows OS and using Horizon Workspace 17 Pro
